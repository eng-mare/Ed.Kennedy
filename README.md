# Edifício Kennedy — Visualizador BIM

Visualizador 3D interativo das disciplinas de projeto do Condomínio Edifício Kennedy.

**Acesso:** [https://seuusuario.github.io/kennedy-bim](https://seuusuario.github.io/kennedy-bim)

## Disciplinas

| Arquivo IFC | Disciplina |
|---|---|
| `KENN-HID-DC-0001-MODE-R00.ifc` | Hidráulico (Barrilete + Prumadas) |
| `KENN-ARQ-DC-0001-MODE-R00.ifc` | Arquitetônico (referência) |

## Como adicionar os modelos

1. Exporte os modelos IFC com os nomes acima
2. Coloque os arquivos na pasta `models/`
3. Faça o commit e push — o visualizador carrega automaticamente

> Arquivos `.ifc.gz` (compactados) também são suportados para reduzir o tempo de carregamento.
